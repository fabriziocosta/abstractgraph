# abstractgraph

Extraction scaffold for the core AbstractGraph package.

This package contains the abstract graph data model, hashing and label helpers,
operators, XML serialization, vectorization, display utilities, preprocessing,
and graph adapters.

Source layout:
- `src/abstractgraph/`
- `docs/README.md`
- `docs/OPERATORS.md`
- `docs/PREPROCESSOR.md`
- `docs/WHITE_PAPER.md`

Local validation:
- `python -m pip install -e . --no-deps`
- `python scripts/smoke_test.py`
